Stack-js
=====
[![npm version](https://badge.fury.io/js/stack-js.png)](https://www.npmjs.com/package/stack-js)

Stack Data Structure implementation in Javascript.

###Install
```
npm install stack-js
```

###Initialize an stack :
```
var myStack = new Stackjs();
```

###Add item :
```
myStack.push(data);
```

###Remove item :
```
myStack.pop();
```

###Peek item :
```
myStack.peek();
```

###Clear stack :
```
myStack.clear();
```

###Length of stack :
```
myStack.length();
```
